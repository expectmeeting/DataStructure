# 临界资源访问

- 临界资源：一次仅允许一个进程/线程使用的资源。
- 临界区：访问临界资源的那段代码。

linux和window下进程和线程的通信手段：

linux-process：管道、信号、消息队列、共享内存、信号量、套接字

window-process:管道、消息队列、共享内存、信号量、套接字

linux-thread：互斥量、信号量、条件变量

window-thread：互斥量、信号量、条件变量、临界区
